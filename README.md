# Irony (Tekken Trainer) ü•ä

A powerful diagnostic tool for **Tekken 8** and **Tekken 7**. See hitboxes, frame data, and move properties in real-time to master your gameplay.

## ‚ú® Features
*   **Visual Hitboxes**: See hit lines, hurt cylinders, and collision spheres.
*   **Live Frame Data**: Measure startup, active, and recovery frames instantly.
*   **Recording & Analysis**: Capture situations and play them back frame-by-frame.
*   **Measure Tools**: Precisely check attack range, distance, and angles.
*   **Multi-View**: Examine the action from front, top, or side angles.

---

## üöÄ Quick Start (Windows)
1.  **Download**: Get the latest version from [Releases](https://github.com/tomislav-ivankovic/Irony/releases/latest).
2.  **Extract**: Unzip the folder anywhere on your PC.
3.  **Run**: Open `irony_injector.exe`.
4.  **Play**: Launch Tekken. Press **`Tab`** in-game to open the menu.

## üêß Linux (Steam Deck / Proton)
